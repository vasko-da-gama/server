# server

Сервер принимает запросы от клиентов и отвечает на них информацией о компьютере на котором запущен. Передаваемая информация шифруется алгоритмом AES128 (https://github.com/vasko-da-gama/My-AES128). Перед шифрованием программа производит небольшие преобразования стандартного ключа с помощью введенной маски во время запуска программы. Такое же значение маски необходимо ввести в программе-клиенте.

Запуск программы: iosrv.exe port key_mask

в случае игнорирования второго и третьего аргумента, будут взяты стандартные значения порта и маски ключа.

## пример вывода

```
:: Server ::
> Waiting...
> Client 127.0.0.1 connected 184
> send to the 184 256bytes
> Client 184 disconected
```
